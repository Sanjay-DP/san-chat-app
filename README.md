# My-chat-app
Developed a high-performance, real-time communication platform using React.js and Firebase. Engineered instant messaging capabilities with Firestore Snapshots for message delivery. Implemented secure user authentication and a responsive UI to provide a seamless chat experience across mobile and desktop devices.
# 💬 Real-Time Chat Application

A modern real-time chat application built using React, Firebase, and Cloudinary.
This project enables users to communicate instantly with features like image sharing, message reactions, and user presence.

## 🚀 Live Demo
gh repo clone Sanjay-DP/san-chat-app

## 📌 Features

* 🔐 User Authentication (Signup/Login)
* 👤 Profile Setup with Avatar Upload
* 💬 Real-time Messaging
* 🖼️ Image Sharing (Cloudinary Integration)
* ✅ Message Seen Status (Double Tick)
* 🗑️ Delete Messages with Confirmation
* 😀 Message Reactions (WhatsApp-style popup)
* 🟢 Online/Last Seen Status
* 📱 Responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, CSS, HTML
* **Backend:** Firebase Firestore (Database)
* **Authentication:** Firebase Auth
* **Storage:** Cloudinary (for images)

## ⚙️ Installation & Setup

Clone the repository:

```bash
gh repo clone Sanjay-DP/My-chat-app
cd My-chat-app
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your Firebase config:

```env
VITE_API_KEY=your_key
VITE_AUTH_DOMAIN=your_domain
VITE_PROJECT_ID=your_project_id
VITE_STORAGE_BUCKET=your_bucket
VITE_MESSAGING_SENDER_ID=your_id
VITE_APP_ID=your_app_id
```

Run the project:

```bash
npm run dev
```
